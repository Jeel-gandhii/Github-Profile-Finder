🔍 GitHub Profile Finder

A simple React JS application where users can enter a GitHub username and instantly see how many public repositories that user has created, along with basic profile information.

🚀 Features
Search any GitHub user by username
Display:
Profile picture
Name
Bio
Followers / Following
Number of public repositories
Clean and responsive UI
Real-time data using GitHub API
🛠️ Built With
React JS
CSS
GitHub REST API
Fetch API / Axios
📸 How It Works
User enters a GitHub username

App calls GitHub API:

https://api.github.com/users/{username}
Displays the user information on the screen
